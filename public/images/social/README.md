# Icons Directory

## Required Icons

Place technology/skill icons in this directory:

### Frontend
- `flutter.png` - Flutter logo
- `react.png` - React logo
- `typescript.png` - TypeScript logo

### Backend
- `nodejs.png` - Node.js logo
- `python.png` - Python logo
- `firebase.png` - Firebase logo
- `postgresql.png` - PostgreSQL logo

### Tools
- `docker.png` - Docker logo
- `aws.png` - AWS logo
- `figma.png` - Figma logo

## Icon Guidelines

1. **Format**: PNG with transparent background
2. **Size**: 128x128px or 256x256px
3. **Style**: Consistent style across all icons
4. **Quality**: High resolution for sharp display

## Free Icon Sources

- [Simple Icons](https://simpleicons.org/) - Brand icons
- [DevIcon](https://devicon.dev/) - Developer icons
- [IconScout](https://iconscout.com/) - Free icon library
- [Flaticon](https://www.flaticon.com/) - Icon collection

## Note

You can also use Material Icons instead of custom icons by setting `"type": "material"` in the JSON configuration.
